# Playwright Pytest Automation Framework

## Overview

This project is a Test Automation Framework built using Python, Playwright, and Pytest. The framework follows the Page Object Model (POM) design pattern to improve maintainability, scalability, and reusability of automated test scripts.

The framework is designed to automate Web UI and API test scenarios and follows industry-standard automation practices.

## Installation

Clone the repository

```bash
git clone https://github.com/Narasapuram-LekhaSudheshna/Playwright-Pytest-Automation-Framework.git
```

Navigate to project folder

```bash
cd Playwright-Pytest-Automation-Framework
```

Install dependencies

```bash
pip install -r requirements.txt
```

Install Playwright browsers

```bash
playwright install
```

---

## Running Tests

Run all tests

```bash
pytest
```

Run tests in headed mode

```bash
pytest --headed
```

Generate HTML report

```bash
pytest --html=reports/report.html
```
